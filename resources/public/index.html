Running your own draft

Do you know how to use curl? I sure hope so. Here's an example command:

curl -v -X POST epicdraft.club/drafts/new \
  -d set-names=Set\ 1,Uprising,Tyrants \
  -d player-count=5 \
  -d player-names='Will Morgan,James,Atlanta,Richard,Jason' \
  -d pack-size=12

The output of that command will be a list of URLs you can send to everyone that you're drafting with.

With great power comes great responsibility! Be smart and use this site responsibly.
